itg_flat Theme for VIM

31 Jan 2015
  • vim
  • theme

Port of the awesome Sublime Text theme itg.flat with some minor tweaks.

Download itg_flat.vim

Installation

Place itg_flat.vim into your .vim/colors/ folder then, add this to your .vimrc file

colorscheme itg_flat
